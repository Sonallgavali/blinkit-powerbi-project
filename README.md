# Blinkit Sales & Operations Analytics Dashboard

## Project Overview
This project is a Power BI dashboard created to analyze Blinkit's sales, customer behavior, product performance, delivery efficiency, and marketing effectiveness.

## Dashboard Pages
1. Sales Overview – Revenue, Orders, Profit, and Category-wise analysis
2. Customer Analysis – New vs Repeat customers and revenue contribution
3. Product Analysis – Top and low-performing products with drill-through
4. Delivery Performance – On-time vs delayed deliveries
5. Marketing Analysis – Campaign performance and promotional impact

## Key Features
- DAX Measures for KPIs
- Drill-through from Category to Product level
- Time slicer using Date table
- Interactive and business-focused visuals

## Tools Used
- Power BI
- DAX
- Data Modeling
